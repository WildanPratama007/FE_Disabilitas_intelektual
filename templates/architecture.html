Arsitektur Cloud dan Machine Learning

Infrastruktur scalable untuk sistem AI deteksi disabilitas intelektual

Cloud Infrastructure Architecture

Komponen Cloud Infrastructure:

  • AWS Lightsail – App Container: Kontainer web/API yang diakses pengguna.
  • PostgreSQL Database (Lightsail): Penyimpanan data aplikasi, terhubung langsung ke App Container.
  • Amazon S3 Storage: Penyimpanan objek untuk input/output (unggah data dari app, ambil hasil prediksi).
  • AWS Lambda – Predict Function: Fungsi yang memanggil proses inferensi dan mengorkestrasi request ke SageMaker.
  • Amazon SageMaker: Endpoint/model ML untuk menjalankan prediksi.
  • Client Access & Routing: Users ⇢ App Container (request), App Container ⇄ S3; alur S3 ⇢ Lambda ⇢ SageMaker ⇢ S3.

Machine Learning Pipeline Architecture

Komponen ML Pipeline:

  • Data Source: Variant Dataset
  • Storage & Preprocessing: Amazon S3 (Data Preprocessing)
  • Model Training: Amazon SageMaker Notebook
  • Model Serving/Deployment: Amazon SageMaker Endpoint
  • Inference Input: Client Input Data (.bed) → S3 (trigger event)
  • Inference Orchestration: AWS Lambda Function (AI Predict)
  • Prediction Output: S3 (hasil prediksi .json)
  • Client Application: Client Web App (mengonsumsi hasil)
  • Monitoring: Amazon CloudWatch (pipeline monitoring)
  • Security & Access: AWS IAM (access control & permissions)
